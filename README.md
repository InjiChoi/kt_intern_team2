## 📍 서비스 소개
1. <b>졸음 인식</b> : 출력된 본인 얼굴의 졸음 상태를 인식
2. <b>집중력 강화 운동</b> : 졸음 인식 후, 집중력 강화 운동을 통해 Refresh
3. <b>수강생 집중도 평가</b> : 졸음의 횟수 등을 파악하여 집중도를 시각화한 데이터 출력

   ➡ <b>단순 강의 학습 도우미 역할 뿐만 아니라 학교 및 기관 차원에서 학습 데이터 수집을 통한 평가 가능</b>
